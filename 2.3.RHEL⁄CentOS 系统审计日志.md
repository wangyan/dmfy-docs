# RHEL/CentOS 系统审计日志

## 一、Rsyslog 日志

### 1.1. OpenSSH 日志

> http://crazy123.blog.51cto.com/1029610/791583

日志格式：`facility.level action`

```shell
vim /etc/ssh/sshd_config
SyslogFacility  local1

vim /etc/rsyslog.conf
local1.* /var/log/sshd.log

systemctl restart rsyslog.service
```

### 1.2. 内核日志

```shell
vim /etc/rsyslog.conf
kern.*                                            /var/log/kern.log
```

设置权限

```shell
chmod 400 /etc/rsyslog.conf
vim /etc/logrotate.conf
weekly
rotate 30
```

## 三、使用 auditd 审计工具

```bash
yum install -y audit

# centos7
cp /usr/share/doc/audit-2.8.5/rules/30-stig.rules /etc/audit/audit.rules

# rhel 8
cp /usr/share/doc/audit/rules/30-stig.rules /etc/audit/audit.rules

service auditd start && chkconfig auditd on

systemctl enable auditd.service
systemctl start auditd.service
```
