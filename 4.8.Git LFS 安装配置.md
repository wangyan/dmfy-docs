# Git LFS 安装配置

## 安装

<https://git-lfs.github.com/>

```bash
wget https://github.com/git-lfs/git-lfs/releases/download/v2.8.0/git-lfs-linux-amd64-v2.8.0.tar.gz
tar -zxf git-lfs-linux-amd64-*
cd git-lfs-linux-amd64-*
git init
sudo ./install.sh
```

## 项目初始化

```bash
git lfs install
git lfs track "*.psd"
git add .gitattributes
```

## 全局配置

<http://shuhrat.github.io/programming/git-lfs-tips-and-tricks.html>

vim ~/.gitconfig

```bash
[filter "lfs"]
	smudge = git-lfs smudge -- %f
	process = git-lfs filter-process
	required = true
	clean = git-lfs clean -- %f
```
